# 罗杰疑案：双重叙述

### After Mr. Ackroyd

> “谁在说谎，谁在调查。”

《罗杰疑案：双重叙述》是一款基于经典侦探叙事结构打造的互动悬疑游戏。项目以“不可靠叙述”为核心，通过碎片化记忆、心理暗示与动态文本演出，让玩家在调查过程中逐渐失去对“真相”的确定感。

游戏整体采用复古像素风格与英式庄园视觉语言，围绕压抑、克制而持续扩散的悬疑氛围展开。随着调查深入，玩家将不断接触到异常记忆、失真的描述以及无法解释的细节，而这些内容会逐渐瓦解原本稳定的叙事结构。

项目重点并不在于传统推理中的“寻找凶手”，而在于玩家如何在调查过程中逐渐意识到：

自己或许并不是单纯的旁观者。
